# Atask
Este é um projeto acadêmico desenvolvido para a disciplina de DevOps
O principal intuito deste repositório é aplicar na prática os conceitos de:
    Controle de Versão: Uso de Git e branches para organizar o código.
    CI/CD: Implementação de automação com GitHub Actions para testar e entregar o software.
    Conteinerização: Uso de Docker para rodar a aplicação em qualquer lugar.
